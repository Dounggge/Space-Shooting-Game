# Savior

A minimalist **2D space shooter prototype** built with Unity.  
Survive 3 stages, dodge bullets, and destroy the final boss.

> No scoring, no upgrades, no resource collection – just pure arcade shooting.

## Gameplay

- **3 handcrafted stages** with increasing difficulty  
- **4 enemy types** – Scout, Fighter, Bomber, Tank (mini-boss)  
- **1 final boss** – Armored Cruiser with 2 attack phases  
- **3 lives** – collision grants brief invincibility (1.5s), game over on 0

## Controls


- Move: `WASD` or Arrow keys 
- Shoot: Hold `Space` or `Left Mouse`
